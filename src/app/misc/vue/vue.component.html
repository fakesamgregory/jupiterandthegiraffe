Vue

(Taken from their website). Vue (pronounced /vjuÀê/, like view) is a progressive framework for building user interfaces. Unlike other monolithic frameworks, Vue is designed from the ground up to be incrementally adoptable. The core library is focused on the view layer only, and is easy to pick up and integrate with other libraries or existing projects. On the other hand, Vue is also perfectly capable of powering sophisticated Single-Page Applications when used in combination with modern tooling and supporting libraries.

Vue is an open-source framework for building front-end JavaScript applications. Its benefits over something like Angular is its lightweight size and easy adoption rate. It was built by an ex-Google engineer and has grown rapidly in popularity over the last few years.
