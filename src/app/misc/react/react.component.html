React

(Taken from their website). React makes it painless to create interactive UIs. Design simple views for each state in your application, and React will efficiently update and render just the right components when your data changes.

React is a front-end JavaScript library for building component-based JavaScript web applications. Its wide adoption rate and lightweight library means it's heavily used in many front-end applications. To build a web application into something more than just a component-based single-page application, you must also use libraries to extend Reacts functionality.

React is built and maintained by Facebook.
