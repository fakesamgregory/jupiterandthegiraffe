UX (User Experience)

User Experience (or UX) is a core part to website development. UX strips away all of the design and focuses on the user journey's. This helps optimise those journey's and make your website or app as easy to use as possible.

We always think from the users perspective in any piece of work we do. The users are established during the Strategy phase of the project where we outline 'user profiles'
